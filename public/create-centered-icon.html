Instructions:

  1. Open your image editor (Photoshop, GIMP, Figma, Canva, or any image editor)
  2. Create a new square canvas:
    • Size: 512x512 pixels (or 1024x1024 for high quality)
    • Background: White (#FFFFFF) or transparent
  3. Add your logo:
    • Place your logo in the center
    • Leave approximately 10-15% padding on all sides
    • For a 512x512 canvas: Logo should be ~360x360 pixels, with ~76px padding
  4. Export as PNG:
    • Save as logo.png
    • Replace the existing file in public/logo.png
  5. Clear cache and re-add to home screen:
    • Clear browser cache
    • Remove the app from home screen (if already added)
    • Re-add to home screen

Quick Reference Sizes:

  • iOS: 180x180 pixels (apple-touch-icon)
  • Android Small: 192x192 pixels
  • Android Large: 512x512 pixels
  • Recommended Canvas: 512x512 or 1024x1024 (can be scaled down)
